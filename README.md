# TSARS
### Tamil Nadu Smart Alcohol Regulation System

> **"Smart Regulation for a Healthier Tamil Nadu."**

TSARS is an AI-powered smart alcohol regulation and public-health monitoring platform designed to encourage responsible alcohol consumption through digital governance, AI analytics, biometric verification, and intelligent monitoring systems.

---

## 🎯 Project Focus

- ✅ Responsible alcohol consumption
- ✅ Addiction reduction
- ✅ Youth protection
- ✅ Smart public-health regulation
- ✅ AI-driven governance infrastructure

> ⚠️ **TSARS is not designed as an alcohol prohibition system.**  
> Instead, the platform aims to create a safer and healthier alcohol ecosystem using modern technology and data-driven governance.

## ✨ Features

### Core Regulation Features

| Feature | Description |
|---|---|
| 🪪 **Digital Consumer Verification** | Aadhaar-based registration with secure identity verification and OTP authentication |
| 🧠 **AI-Based Risk Detection** | AI analytics engine identifies suspicious buying patterns and high-risk alcohol consumption behavior |
| 📊 **Government Analytics Dashboard** | Real-time monitoring of total alcohol sales, unique buyer count, district-wise consumption trends, and addiction-risk zones |
| ⏳ **Cooling Period System** | Mandatory waiting period for newly registered users to reduce impulsive alcohol consumption |
| 🍾 **Smart Purchase Limit System** | Age-based monthly purchase limits with weekly distribution control |
| 🔒 **Privacy & Security Focused** | Encrypted storage, restricted access, and regulation-oriented implementation |
| 🚫 **Anti-Black-Market Monitoring** | Detection of suspicious resale behavior, identity misuse, and bulk purchase attempts |
| 🏥 **Public Health Awareness Integration** | Health-awareness messages, addiction-support resources, and responsible-consumption education |
| 📱 **Modern Digital Platform** | Scalable architecture designed for mobile apps, admin dashboards, and AI analytics systems |

## 🏗️ System Architecture

```
Consumer
   ↓
TSARS Seller Application
   ↓
Verification System
   ↓
Central Regulation Server
   ↓
AI Monitoring Engine
   ↓
Government Analytics Dashboard
```

### Main Modules

| Module | Description |
|---|---|
| `tsars-seller` | TASMAC seller-side application |

## ⚙️ Core Workflow

### Step 1 — Consumer Registration
- Aadhaar verification
- Mobile OTP authentication
- Face/Fingerprint verification

### Step 2 — Cooling Period
- Mandatory waiting period for first-time users

### Step 3 — Purchase Validation
- Monthly eligibility check
- Purchase-limit verification
- AI-based risk detection

### Step 4 — Analytics & Monitoring
- Real-time sales tracking
- Buyer analytics
- Addiction-risk analysis
